====== about ======

Здесь собран набор подсказок/инструкций для установки, переустановки и настройки NAS под OMV. Если кому-то это всё тоже будет полезно — велком.

* NAS — Network Attached Storage
* OMV — OpenMediaVault, основан на Debian.

Местами подсказки обширные, на уровне объяснения. Если сам себе всё понятно объяснил, значит — понял. Когда ошибся — переписываю, иногда чуть менее, чем полностью.

===== Навигация =====

Выстроена через индекс страниц — https://testitquickly.com/bystro.linux.nas/

===== Как открыть «bystro.linux» в wiki-редакторе Zim =====

==== Снять файлы блокнота из git ====

https://github.com/testitquickly/bystro.linux.nas/

==== Открыть в Zim файл „notebook.zim” ====

Можно сразу добавить новый блокнот во внутренний список блокнотов Zim:

''Открыть Zim''
''> File''
''> Open Another Notebook (Ctrl+O)''
''> Add''
''> [Browse]''
''> Указать каталог с этими файлами.''

Подхватится файл notebook.zim, блокнот откроется.
