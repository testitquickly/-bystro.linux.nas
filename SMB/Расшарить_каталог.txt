====== Расшарить каталог ======

Шарить через SMB можно только те shared-каталоги, которые уже были добавлены (зарегистрированы) в OMV. См. [[Shared Folders:Добавить shared-каталог]]

''Services'' 
''> SMB/CIFS'' 
''> Shares'' 
''> [+ Add]''

	* Enable = Да
	* Shared folder = выбрать предлагаемый из списка \tire \textit{test}
	* Comment= очень нужен, когда много каталогов
	* Public = No
	* Read only = No
	* Browseable = Да
	* Inherit ACLs = Да
	* Hide dot files = Да

Остальное не трогаем. 

Со смартфона проверить появление каталогов на NAS для каждого пользователя по-отдельности можно через приложение «Cx Проводник» (оно же Cx File Explorer).
